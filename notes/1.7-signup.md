## Create a signup form

* Create the `signup.html` file in your working directory and add a link in your `login.html` under login button
```
<!-- login.html -->
<h6>New to Web Tweet? <a href="signup.html">Sign up Now</a></h6> 
```
```
/* main.css */
h6 {
    font-weight: 300;
    color: #B4B4B4;
    margin: 1rem 0;
    font-size: .83rem;
}
```
